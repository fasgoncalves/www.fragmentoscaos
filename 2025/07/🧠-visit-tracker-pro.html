🚀 O que é o Visit Tracker Pro?

O Visit Tracker Pro é um plugin WordPress desenvolvido para quem quer monitorizar e compreender o comportamento dos visitantes de forma clara, eficaz e visual.
Totalmente personalizável e leve, oferece uma alternativa prática e sem dependências externas a soluções invasivas como Google Analytics.

🛠️ Funcionalidades principais

🧠 Como funciona?

O plugin regista de forma automática:

  • O IP do visitante
  • A página visitada
  • O user agent (navegador)
  • A hora e sessão

Todos os dados ficam armazenados na base de dados local (MySQL), e são apresentados no painel com Chart.js — elegante, responsivo e interativo.

🔐 Privacidade em primeiro lugar

  • ✅ Não usa cookies
  • ✅ Não envia dados para terceiros
  • ✅ Totalmente compatível com GDPR

💡 Para quem é o Visit Tracker Pro?

  • 🔹 Bloggers que querem saber quais artigos têm maior impacto
  • 🔹 Pequenas empresas que precisam de dados sem entregar tudo ao Google
  • 🔹 Admins técnicos que desejam estatísticas sem tracking externo
  • 🔹 Equipas criativas que valorizam interface limpa e gráficos intuitivos

⚙️ Requisitos mínimos

  • WordPress 5.5+
  • PHP 7.4+
  • Chart.js incluído automaticamente
  • Base de dados MySQL ativa (usa prefixo padrão)

📦 Disponibilidade

O Visit Tracker Pro está disponível em versão .zip para instalação direta no WordPress.
Funciona como um plugin independente ou pode ser expandido com módulos de IA como o Tracker Insight AI.
